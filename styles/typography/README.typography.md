# Typography System Architecture

**TYPOGRAPHICAL ARMAGEDDON WARNING**: This directory contains the typography system that controls every piece of text on your website. These files manage font loading, Figma design token integration, and Tailwind utility generation. Mess with the wrong file or ignore the loading order and watch your carefully crafted typography degrade into a typographical disaster that looks like it was designed by someone who thinks Comic Sans is a legitimate business font.

## Architecture Overview (The Typography Command Center)

The typography system is a precisely orchestrated combination of:

- **Figma Integration**: Auto-generated font families from `figma/services/TypographyService.js`
- **Font Loading**: Strategic font imports with performance optimization
- **Tailwind Utilities**: Generated font-family classes for consistent typography
- **Loading Order**: CRITICAL dependency chain that breaks if violated

**ABSOLUTE CRITICAL RULE**: Font imports MUST load before Tailwind CSS compilation. Violate this order and watch your font-family utilities reference fonts that don't exist, causing a cascade of fallback font chaos.

## Font Import Management (`imports.css`)

### Purpose & Responsibility

**What it controls**: All font loading for the entire design system
**Load priority**: HIGHEST - must load before any other CSS processing
**Performance impact**: CRITICAL - affects page load speed and font display

### Font Import Patterns

**Google Fonts Integration**:

```css
/* Strategic font loading with display optimization */
@import url("https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap");
@import url("https://fonts.googleapis.com/css2?family=JetBrains+Mono:wght@400;500&display=swap");
```

**Custom Font Loading**:

```css
/* Self-hosted fonts with proper fallbacks */
@font-face {
  font-family: "Custom Brand Font";
  src:
    url("/assets/fonts/brand-font.woff2") format("woff2"),
    url("/assets/fonts/brand-font.woff") format("woff");
  font-display: swap;
  font-weight: 400;
}
```

**PERFORMANCE CRITICAL RULES**:

- **Always use `font-display: swap`**: Prevents invisible text during font load
- **Specify font weights**: Only load weights actually used in design
- **Use WOFF2 first**: Modern format loads faster than legacy formats
- **Provide fallbacks**: System fonts prevent layout shift during font load

### Font Display Strategy

**`font-display: swap` Benefits**:

- **Immediate text visibility**: Shows fallback font immediately
- **Progressive enhancement**: Swaps to web font when loaded
- **Performance optimization**: No blocking of text rendering
- **User experience**: Content readable during slow font loads

## Font Loading Performance Optimization

### Critical Loading Path

**Optimized Loading Sequence**:

1. **Font imports load first** via `imports.css`
2. **Browser begins font download** immediately
3. **Fallback fonts display content** while web fonts load
4. **Font swap occurs** when web fonts complete loading
5. **Generated utilities reference loaded fonts** via `fontFamilies.css`

### Performance Monitoring

**Font Loading Metrics to Track**:

- **First Contentful Paint (FCP)**: Text appears with fallback fonts
- **Largest Contentful Paint (LCP)**: Major text elements render
- **Cumulative Layout Shift (CLS)**: Font swap causes minimal layout shift
- **Font Load Time**: Web fonts complete downloading

### Optimization Strategies

**Strategic Font Subsetting**:

```css
/* Load only required character sets */
@import url("https://fonts.googleapis.com/css2?family=Inter:wght@400;600&text=abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789&display=swap");
```

**Font Preloading for Critical Text**:

```html
<!-- Preload fonts for above-the-fold content -->
<link
  rel="preload"
  href="/assets/fonts/heading-font.woff2"
  as="font"
  type="font/woff2"
  crossorigin
/>
```

**Resource Hints**:

```html
<!-- DNS prefetch for external font resources -->
<link rel="dns-prefetch" href="//fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
```

## Development Workflow (Safe Typography Operations)

### Adding New Fonts

Step 1: Add Font Import

```css
/* Add to imports.css */
@import url("https://fonts.googleapis.com/css2?family=New+Font:wght@400;700&display=swap");
```

Step 2: Create Figma Text Style

- Create new text style in Figma Typography frame
- Set font-family to match imported font name
- Follow naming convention: `Category/Weight` (e.g., `Accent/Bold`)

Step 3: Sync Design Tokens

```bash
npm run build:design  # Regenerates fontFamilies.css with new utilities
```

Step 4: Test Font Loading

- Verify font loads correctly in browser
- Check fallback font behavior during load
- Test on slow network connections

### Updating Existing Fonts

**Safe Update Process**:

1. **Update font import** in `imports.css` (change weights, add styles)
2. **Update Figma text styles** to match new font specifications
3. **Regenerate utilities** with `npm run build:design`
4. **Test thoroughly** across all font usage patterns

**NEVER**:

- Edit `fontFamilies.css` manually (changes will be overwritten)
- Change font loading order in `main.css` (breaks cascade)
- Remove fonts from imports without updating Figma styles

### Debugging Typography Issues

**Common Problems & Solutions**:

**"Font not loading/displaying"**:

- **Check import syntax** in `imports.css` for typos
- **Verify network access** to Google Fonts or custom font URLs
- **Test fallback fonts** work correctly during font load
- **Check browser developer tools** for font loading errors

**"Font utility classes not working"**:

- **Run design token sync**: `npm run build:design`
- **Check Figma text style structure** matches expected hierarchy
- **Verify font names match** between imports and Figma styles
- **Test Tailwind compilation** includes generated utilities

**"Layout shifting during font load"**:

- **Use `font-display: swap`** in all font declarations
- **Match fallback font metrics** to web font dimensions
- **Consider font loading strategies** like font-display: optional
- **Test on slow connections** to identify timing issues

**"Generated utilities missing fonts"**:

- **Check TypographyService.js logs** for parsing errors
- **Verify Figma API token** has access to design file
- **Test Figma text style structure** follows exact naming convention
- **Regenerate tokens** after fixing Figma structure issues
