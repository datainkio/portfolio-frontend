# Motion Checklist

## Scroll Behavior

- [ ] pin durations feel intentional
- [ ] no abrupt releases
- [ ] no overlapping triggers
- [ ] scroll pacing remains readable

## Performance

- [ ] no dropped frames
- [ ] no Safari repaint artifacts
- [ ] smooth inertial scrolling

## Narrative

- [ ] transitions support hierarchy
- [ ] motion does not compete with content
- [ ] visual rhythm remains coherent
