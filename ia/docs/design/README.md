This directory is the design-system reference area for component behavior and composition rules.

## Atomic Design Role

It is the umbrella for all atomic tiers, from indivisible primitives to page-level composition.

## Subsections

- `atoms/`: base visual and semantic primitives.
- `molecules/`: small compositional patterns.
- `organisms/`: section-level systems.
- `templates/`: page pattern blueprints.
- `layouts/`: structural shells and shared page framing.
